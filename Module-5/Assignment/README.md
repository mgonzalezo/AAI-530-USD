# Smart Home Energy Monitoring Dashboard
## Module 5 Assignment - AAI-530

Interactive IoT dashboard with ML-powered predictions for household energy consumption monitoring.

## Features

This Streamlit dashboard includes all required visualizations:

### Summary/Status Visualizations:
1. **Current Power Status** - Large display showing current power consumption with color-coded usage categories
2. **Daily Usage Summary** - Bar chart showing average daily power consumption over 14 days
3. **Alert Count Status** - KPI display showing number of high-power alerts (>5 kW)
4. **Circuit Breakdown** - Pie chart showing power distribution across sub-metering circuits

### Machine Learning Visualizations:
5. **Prediction vs Actual** - Dual-line chart comparing actual power consumption with ML predictions
6. **Prediction Error Distribution** - Histogram showing the accuracy of ML model predictions

### Interactive Features:
- Date range filter
- Usage category filter
- Hover tooltips on all charts
- Responsive layout
- Real-time status indicators

## Local Setup

### Prerequisites
- Python 3.8 or higher
- pip package manager

### Installation

1. Navigate to the assignment directory:
```bash
cd /home/margonza/Documents/Marco/Master/AAI-530-IN1/AAI-530-USD/Module-5/Assignment
```

2. Install required packages:
```bash
pip install -r requirements.txt
```

### Running the Dashboard

Run the Streamlit app:
```bash
streamlit run dashboard.py
```

The dashboard will open in your default browser at `http://localhost:8501`

## Deployment to Streamlit Cloud

### Steps:

1. **Commit and push to GitHub** (if not already done)

2. **Go to Streamlit Cloud**:
   - Visit https://streamlit.io/cloud
   - Sign in with your GitHub account

3. **Deploy the app**:
   - Click "New app"
   - Select your repository: `mgonzalezo/AAI-530-USD`
   - Branch: `main`
   - Main file path: `Module-5/Assignment/dashboard.py`
   - Click "Deploy!"

4. **Get your public URL**:
   - After deployment, you'll get a URL like: `https://[your-app-name].streamlit.app`
   - This URL can be shared publicly

## Exporting as PDF

### Method 1: Browser Print (Recommended)
1. Open the dashboard in your browser
2. Press `Ctrl+P` (or `Cmd+P` on Mac)
3. Select "Save as PDF" as the destination
4. Adjust settings:
   - Layout: Landscape (recommended)
   - Paper size: A3 or Tabloid (for better visibility)
   - Margins: None
   - Background graphics: Enabled
5. Save as `Energy_Monitoring_Dashboard.pdf`

### Method 2: Screenshot Tool
1. Use a full-page screenshot tool (e.g., browser extensions like "GoFullPage")
2. Capture the entire dashboard
3. Convert to PDF if needed

## Design Elements

### Color Scheme (Pre-attentive Attributes):
- **Blue (#3498DB)**: Actual values, normal usage
- **Orange (#F39C12)**: Predictions, high usage
- **Red (#E74C3C)**: Alerts, very high usage
- **Green (#2ECC71)**: Success status, low usage

### Layout:
- **Top Row**: Status at a glance (current power, alerts, circuit breakdown)
- **Middle Row**: Trend analysis (daily consumption patterns)
- **Bottom Row**: ML performance (predictions vs actual, error distribution)
